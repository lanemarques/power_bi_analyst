 📊 Dashboard de Vendas - Power BI

## 📌 Descrição

Projeto desenvolvido como parte do desafio da DIO, com foco na criação e replicação de dashboards no Power BI utilizando uma base de dados de vendas.

## 🎯 Objetivo

* Reproduzir duas páginas de relatório apresentadas em aula
* Desenvolver uma terceira página com visuais adicionais para análise de dados

## 🛠️ Ferramentas utilizadas

* Power BI
* GitHub

## 📈 Estrutura do Dashboard

### 🔹 Página 1

Relatório replicado conforme apresentado no curso, contendo análise geral de vendas.

### 🔹 Página 2

Relatório replicado com foco em segmentação e indicadores complementares.

### 🔹 Página 3 (Desenvolvida)

Página criada com os seguintes visuais:

* 🌍 **Mapa 1:** Soma de vendas (Sales) e unidades vendidas por país
* 🌍 **Mapa 2:** Soma de lucro (Profit) por país
* 🥧 **Gráfico de pizza:** Lucro por segmento

## 📊 Diferenciais aplicados

* Organização e alinhamento dos visuais
* Renomeação clara dos gráficos
* Configuração de tooltips (dicas de ferramenta)
* Melhor legibilidade das informações
